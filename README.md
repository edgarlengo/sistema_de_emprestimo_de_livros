Sistema de Empréstimo de Livros.

Este é um sistema web simples para gestão de empréstimo de livros, desenvolvido com tecnologias web básicas, as tecnologias utilizadas na construção do projeto e as funcionalidades
virão listadas a baixo.

 Tecnologias Utilizadas:

- HTML5
- CSS3
- JavaScript
- Navegador Web (Edge)
  
Funcionalidades

-tela de login
- Cadastro de livros
- Cadastro de autores
- Cadastro de leitores
- Empréstimo de livros
- Devolução de livros
- Eliminação de registos

É de salientar que este projeto não utiliza backend, os dados podem ser armazenados temporariamente em Arrays JavaScript, não há persistência em base de dados.
